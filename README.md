# 📊 InsightCart-Analysis  

Data cleaning, trend analysis, and customer insights project analyzing purchase behavior across multiple channels using Excel.  

---

## 🔍 Project Overview  
Businesses need to understand how customers purchase across **Website, App, and In-Store** channels.  
This project analyzes transactional data from **InsightCart** to uncover trends, customer preferences, and sales performance.  

---

## 🎯 Objectives  
- Clean and prepare raw transactional data.  
- Identify **purchase trends over time**.  
- Compare performance of **purchase channels**.  
- Analyze **customer-level behavior** and loyalty.  
- Provide a **case study** of a single customer journey.  

---

## 🛠️ Tools & Skills Used  
- **Excel** → Data Cleaning, Pivot Tables, Charts, Conditional Formatting  
- **Data Analysis** → Trend Analysis, Customer Profiling, Success Rate Calculations  
- **Visualization** → Summarized dashboards and clear tables  

---

## 🔑 Process & Insights  

### 1. Data Cleaning  
✔ Removed duplicates and blanks  
✔ Standardized dates and product names  
✔ Handled missing purchase amounts  

---

### 2. Trend Analysis  
- Aggregated **purchase amounts by date**  
- Found peak transaction periods  
- Example: On **2023-12-02**, total purchases peaked at **₹4140**  

---

### 3. Purchase Mode Analysis  

| Channel   | Total Sales | Transactions | Success % |  
|-----------|-------------|--------------|-----------|  
| Website   | ₹81,070     | 182          | 71.9%     |  
| App       | ₹61,835     | 130          | 73.8%     |  
| In-Store  | ₹56,070     | 105          | 76.1%     |  

✅ Insight: **In-store purchases** had the highest success rate, while **Website** generated the highest revenue.  

---

### 4. Customer Analysis  
- Built pivot tables to analyze **spending per customer**  
- Identified **loyal customers** with repeated transactions  

---

### 5. Case Study: Subbarao  
📌 Example customer with repeated transactions  
📊 Showcased purchasing patterns and loyalty
